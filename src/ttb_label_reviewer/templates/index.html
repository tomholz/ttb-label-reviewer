Review a batch

Upload one zip containing manifest.csv and the label images it names — one row per application. Download the CSV template to start from the expected columns.

Up to 500 rows. In the manifest, list each row's images separated by semicolons (e.g. front.png;back.png); a row with a problem is reported in the results and never stops the rest of the batch.
